Setup
========
Clone repository and create directory ::

    $ git clone git://github.com/muquu/dotemacs.git ~/.emacs.d
    $ mkdir ~/.emacs.d/site-lisp

Byte compile ``inits`` directory ::

    C-u M-x byte-recompile-directory
